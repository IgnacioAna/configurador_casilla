  <action>
Crear un proyecto Vite + React 18 (JavaScript, NO TypeScript) desde el root del proyecto.

1. Crear `package.json` con:
   - `"name": "configurador-impacar"`, `"version": "0.1.0"`, `"private": true`, `"type": "module"`
   - scripts: `"dev": "vite"`, `"build": "vite build"`, `"preview": "vite preview"`
   - dependencies: `"react": "^18.3.1"`, `"react-dom": "^18.3.1"`
   - devDependencies: `"vite": "^5.4.0"`, `"@vitejs/plugin-react": "^4.3.0"`
     (las devDeps de Tailwind se agregan en la Task 2)

2. Crear `vite.config.js`:
   ```js
   import { defineConfig } from 'vite'
   import react from '@vitejs/plugin-react'

   export default defineConfig({
     plugins: [react()],
   })
   ```

3. Crear `index.html` en el root con `lang="es-AR"`, un `<div id="root"></div>`,
   `<title>Configurador Impacar</title>`, y `<script type="module" src="/src/main.jsx"></script>`.

4. Crear `src/main.jsx` que monta React 18 con `createRoot`:
   ```jsx
   import { StrictMode } from 'react'
   import { createRoot } from 'react-dom/client'
   import App from './App.jsx'
   import './styles/index.css'

   createRoot(document.getElementById('root')).render(
     <StrictMode>
       <App />
     </StrictMode>,
   )
   ```
   (El import de `./styles/index.css` apunta al archivo que crea la Task 2 — está bien,
   las dos tasks son del mismo plan y el archivo existirá antes de correr el dev server.)

5. Crear `src/App.jsx` con un placeholder mínimo (se reemplaza en la Task 3):
   ```jsx
   export default function App() {
     return <div>Configurador Impacar</div>
   }
   ```

6. Crear `.gitignore` con al menos: `node_modules`, `dist`, `.DS_Store`, `*.local`.

7. Ejecutar `npm install` para generar `node_modules` y `package-lock.json`.
  </action>
